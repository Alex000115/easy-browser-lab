# Easy Browser Lab

This is a very simple project for beginners.

You can use this repository to:
- Learn how a basic browser project works
- Practice editing files
- Write small notes
- Build a habit of daily GitHub commits

How to use:
1. Open any file in this repository
2. Make a small change
3. Save the file
4. Commit your update

Goal:
Practice every day and improve step by step.

Notes:
Write your learning notes below.
